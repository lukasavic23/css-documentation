# css-documentation
Kingdom CSS Documentation

Best way to write CSS - for everything to be consistent across.

**1. Order of writing css **

Inside of a class, CSS properties should be written in the following order.

**.some-class {

     display: block;
     position:relative;
     left: 10px;
     top: 10px;
     right: 10px;
     bottom: 10px;
     
     width: 10px;
     height: 10px;
     
     margin: 10px;
     padding: 10px;
     
     background-color: black;
     background-repeat: no-repeat;
     border: 1px solid black;
     
     color: white;
     font-family: Arial;
     font-size: 10px;
     font-weight: bold;
     font-style: italic;
     text-align: center;
     text-decoration: none;
     
     border-radius: 10px 10px 10px 10px;
     text-shadow: 10px 10px 5px 10px black;
     box-shadow: 10px 10px 5px 10px black inset;
     
     overflow: hidden;
     z-index: 10;
     cursor: pointer;

**}**
